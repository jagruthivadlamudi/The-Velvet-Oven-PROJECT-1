# The Velvet Oven (CSC 4370/6370 Project 1)

A multi-page static bakery/café website built with **HTML5 + CSS3** (no JavaScript).

## Pages
- `index.html` — Home
- `menu.html` — Menu
- `about.html` — About
- `gallery.html` — Gallery
- `contact.html` — Contact
- `team.html` — Team Introduction (required)

## Features / Requirements Covered
- Semantic HTML structure (header/nav/main/section/footer)
- Responsive design (mobile-first breakpoints)
- CSS Grid + Flexbox layouts
- Transitions + transforms (hover lift/zoom)
- CSS animations (hero fade + specials pulse)
- Accessibility (focus styles + prefers-reduced-motion)

## How to Run
Open `index.html` in a browser, or use VS Code Live Server.
